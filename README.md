# 📝 微信记账小程序

一款简洁优雅的个人记账工具，帮助您轻松管理日常收支，合理规划财务。

![小程序码](https://via.placeholder.com/200x200?text=小程序码)

## ✨ 核心功能

### 📊 智能预算管理
- 设置月度预算，实时监控消费进度
- 可视化进度环，直观展示已用比例
- 智能计算每日可用额度，帮助合理消费

### 💰 快速记账
- 一键"记一笔"，3秒完成记账
- 6个常用分类快捷入口
- 支持支出/收入双类型记录
- 可添加备注和图片，记录更详细

### 📖 记账本
- 按月份查看所有记账记录
- 按日期分组展示，清晰明了
- 左滑编辑/删除，操作便捷
- 支持历史月份回溯查询

### 📈 数据统计
- **分类统计**：饼图展示各类支出占比
- **趋势分析**：柱状图/折线图展示消费趋势
- **排行榜**：快速了解主要消费类别
- **年度对比**：年/月视图自由切换

### 🏦 资金池
- 设立多个储蓄目标（旅行基金、购房基金等）
- 随时存入/取出，灵活管理
- 进度条直观显示目标完成情况
- 年度资金总结，见证财富积累

### 📱 个人中心
- 自定义头像和昵称
- 随时调整月预算
- 查看年度报告
- 数据导出备份

### 📋 年度报告
- 年度消费总览
- 月度消费趋势
- 分类消费排行
- 资金池年度总结
- 支持分享给好友

## 🎯 使用场景

- **日常消费** - 记录每一笔支出，了解钱花在哪里
- **预算控制** - 设定月度预算，避免超支
- **储蓄计划** - 创建资金池，实现储蓄目标
- **财务分析** - 通过图表分析消费习惯，优化支出结构


## 💡 使用技巧

1. **设置预算** - 首次使用先在"我的"页面设置月预算
2. **快捷记账** - 首页的6个快捷分类可以快速记录常见消费
3. **定期查看统计** - 每周查看统计页，了解消费结构
4. **建立资金池** - 为特定目标（如旅行、购物）建立资金池，专款专用
5. **导出备份** - 定期导出数据，防止丢失

## 🛡️ 数据安全

- 所有数据存储在本地，保护隐私
- 支持数据导出，可随时备份
- 不上传任何个人信息到服务器
